# Задание 41

## Отчёт

### case1

Использовать для поиска доменов ключевые параметры вместо `id`. Для проверки совпадения объектов использовать функцию
equals, которую должны по контракту содержать все домены.

### case2

Отделить модели, применяемые в REST API и не использовать их для доменов. При этом понадобятся мапперы, отображающие
модели в домены и наоборот. Хранить домены можно в мапах, где в качестве ключа использовать любое уникальное свойство
домена.
